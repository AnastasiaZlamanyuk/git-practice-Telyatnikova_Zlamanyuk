git version - можно посмотреть версию Git
git config--global user.name "ваш_ник" - можно задать ник
git config--global user.email "ваша_почта@yandex.ru" - можно задать имэйл
cd - переход в папку
mkdir - создание папки
git init - инициализация локального репозитория
git remote add origin ссылка_на_репозиторий - привязка удаленного репозитория
git remote-v - проверка на подключения удалённого репозитория
dotnet new console-o Program--use-program-main
Пояснение команды:
dotnet new console — создаёт новый шаблон консольного приложения.
-o Program — указывает имя папки, в которой будет создан проект.
--use-program-main — создаёт классическую точку входа Main, а не упрощённый вариант с программой верхнего уровня.
git status - проверить состояние репозитория
git add . - добавить файлы в индекс
git commit -m "" - создать коммит
git log - история коммитов
git log --oneline - история коммитов компактнее
git show ваш_хеш - показ отдельного коммита
git checkout-b new-feature - создать новую ветку с переходом на неё
git branch - просмотр всех веток
git switch-c - создание ветки и переключение на неё
git switch - переключение на ветку
git pull - обновление ветки
git fetch--prune - состояние сервера
